# Sales Prediction and Inventory Alert System

This project predicts sales performance using historical retail sales data and machine learning.

The model classifies future sales into three categories:
Low Sales
Medium Sales
High Sales

## Technologies Used
Python
Pandas
NumPy
Scikit-learn
XGBoost
Matplotlib
Seaborn

## Features
Data cleaning and preprocessing
Feature engineering using historical sales
Machine learning model using XGBoost
Sales classification model
Confusion matrix visualization
Stock alert system based on predicted sales

## Goal
Help businesses forecast demand and manage inventory more efficiently.

## Dataset
Retail sales dataset (mrtssales92-present.xls)

## Output
Model accuracy
Classification report
Confusion matrix
Stock alert recommendations
